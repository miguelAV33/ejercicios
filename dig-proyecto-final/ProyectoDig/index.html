Bloque I · Solo CSS/HTML

La Psicología de la Forma

Las leyes de la Gestalt describen cómo el cerebro humano organiza estímulos visuales en patrones coherentes. Las siguientes demostraciones están construidas únicamente con elementos HTML y CSS — sin ninguna imagen externa — y muestran figuras que percibes aunque no estén dibujadas de forma explícita.

Ley de Cierre

El cerebro completa figuras incompletas. A la izquierda verás tres sectores circulares (no un triángulo); a la derecha, un arco abierto (no un círculo). Tu mente cierra ambas formas.

Ley de Proximidad

Los elementos cercanos entre sí se perciben como un grupo. Los nueve puntos rojos de la izquierda se leen como una unidad; los azules de la derecha, separados, se perciben individualmente.

Ley de Continuidad

El ojo sigue la trayectoria más suave. Aunque las dos curvas se cruzan en los puntos naranjas, el cerebro las percibe como dos líneas continuas independientes, no como cuatro segmentos.

Ley de Similitud

Los elementos con la misma forma o color son agrupados perceptualmente. Aunque estén mezclados, agrupas los círculos rojos por un lado y los cuadrados azules por otro.

Laboratorio de Color

Los espacios de color determinan cómo representamos el color en pantalla. Usa el conmutador de temas (barra superior) para explorar cómo cambian todos los colores de esta página entre modo claro, oscuro y alto contraste — gestionados 100% con variables CSS.

🔴 Espacio RGB

RGB (Red, Green, Blue) define cada color sumando luz de tres canales — rojo, verde y azul — con valores de 0 a 255. Es el modelo nativo de las pantallas de ordenador, ya que cada píxel emite esos tres colores.

Ejemplo: rgb(220, 53, 69) produce un rojo intenso. Mezclar los tres canales al máximo da blanco puro: rgb(255, 255, 255).

🌈 Espacio HSL

HSL (Hue, Saturation, Lightness) es más intuitivo para diseñadores: el matiz se especifica en grados (0°–360°) en la rueda de color, la saturación y la luminosidad como porcentaje.

Crear una paleta análoga es tan simple como rotar el matiz: hsl(0, 72%, 51%) → hsl(30, 80%, 52%) → hsl(60, 85%, 46%).

◑ Contraste y WCAG 2.1

Las WCAG 2.1 (Web Content Accessibility Guidelines) exigen un ratio de contraste mínimo de 4.5:1 para texto normal (nivel AA) y 7:1 para nivel AAA. El texto grande y de interfaz requiere al menos 3:1.

✓ PASS AA+AAA – ratio 7.2:1
✓ PASS AA – ratio 4.6:1
✗ FAIL – ratio 1.9:1 (insuficiente)

🎨 Armonía Análoga

Esta web usa una paleta análoga: rojo (0°), naranja-rojo (15°), naranja (30°), naranja-amarillo (45°) y amarillo (60°) como familia cálida, con azul oscuro (210°) como contrapunto frío.

HTML Semántico

Las etiquetas correctas permiten a los lectores de pantalla entender la jerarquía y el propósito de cada sección. Nunca <div> donde corresponde <button> o <nav>.

<main> contenido principal
<nav> navegación
<article> unidad de contenido
<button> acción
<div onclick> ← incorrecto
<footer> pie de página
🎨

discapacidad-visual.css

Hoja de estilos dedicada exclusivamente a la lectura accesible. Cubre: fuente mínima ≥16px, contraste ≥4.5:1 (WCAG AA), interlineado 1.75×, letter-spacing, word-spacing, foco ultra-visible, forced-colors, prefers-contrast, estilos de impresión y zoom hasta 400%.

⌨️

Foco y Navegación con Teclado

Todo elemento interactivo es alcanzable con Tab y activable con Enter. El skip link al inicio del DOM permite saltar directamente al contenido sin pasar por la navegación.

Contraste WCAG AA / AAA

Todos los textos de esta página superan el ratio de contraste 4.5:1 exigido por WCAG 2.1 nivel AA, incluso en modo oscuro. El texto principal alcanza 16.8:1 (nivel AAA) en modo claro.

✓ Texto principal – 16.8:1
✓ Texto secundario – 8.1:1
✗ Ejemplo fallido – 1.9:1
🎤

ARIA – Roles y Etiquetas

Atributos aria-label, aria-pressed, aria-labelledby y role enriquecen la semántica de cada elemento para los lectores de pantalla.

role="progressbar" · aria-valuenow="72"

⏸️

Reducción de Movimiento

La media query prefers-reduced-motion: reduce desactiva todas las animaciones CSS para usuarios con sensibilidad vestibular o epilepsia fotosensible. Definida en discapacidad-visual.css.

@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation: none !important;
    transition: 0.01ms !important;
  }
}
