# 🛒 La Compra — React v3 (Diseño Premium)

## Arrancar ya (sin instalar nada)
```
cd build
python3 -m http.server 3000
```
→ http://localhost:3000

O copia la carpeta `build/` en `C:\xampp\htdocs\lacompra\`
→ http://localhost/lacompra

## Desarrollo
```
npm install
npm start
```

## Interactividad
- **Swipe izquierda** para eliminar un producto
- **Doble clic** sobre el nombre para editar inline
- **Hover** muestra botones de editar y eliminar
- **Confetti** al completar toda la lista
- **Barra de progreso** y **anillo SVG** animados en tiempo real
- **Formulario colapsable** con validación y flash al añadir
- **Header sticky** con blur al hacer scroll
